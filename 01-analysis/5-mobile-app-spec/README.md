## 3. Соглашения

### 3.1. Платформа
- **iOS:** 14.0+
- **Android:** 8.0+
- **Фреймворк:** React Native

### 3.2. API
- **Base URL (Production):** `https://api.vertical-climbing.ru`
- **Base URL (Staging):** `https://staging-api.vertical-climbing.ru`
- **Версия API:** v1
- **Аутентификация:** Bearer Token (JWT)

### 3.3. Числа и параметры
- **Все числа** (лимиты, цены, размеры) **не хардкодятся** в UI.
- Значения приходят из API-контракта.
- Пример: `slot.capacity`, `rental_gear.shoes_price`.
