## 5. Описание логики
Логика формирует Deep Links (или Universal Links) для популярных картографических сервисов. Адрес должен быть URL-encoded.

1. **Яндекс.Карты:** `yandexmaps://maps.yandex.ru/?text={encoded_address}` (Fallback: `https://yandex.ru/maps/?text=...`)
2. **2GIS:** `dgis://2gis.ru/search/{encoded_address}` (Fallback: `https://2gis.ru/search/...`)
3. **Apple Maps (только iOS):** `http://maps.apple.com/?q={encoded_address}`
4. **Google Maps (Android/Web):** `https://www.google.com/maps/search/?api=1&query={encoded_address}`

*Примечание:* Для MVP достаточно использовать универсальные HTTP-ссылки, которые ОС сама предложит открыть в установленном приложении, если оно есть.
