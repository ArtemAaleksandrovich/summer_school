# Фича-лист (Feature List)

**Проект:** Мобильное приложение «Вертикаль» (Клиент)  
**Версия:** 1.0 (MVP)  
**Дата:** 2026-07-04

---

## 1. Назначение

**Продукт:** Мобильное приложение для самостоятельной записи клиентов скалодрома «Вертикаль» на групповые тренировки.

**Скоуп MVP:**
- Клиентское мобильное приложение (iOS + Android)
- API для клиентского приложения
- **Вне скоупа:** Админка, интерфейс инструктора (уже существуют в инфраструктуре)

**Основная ценность:**
- Для клиента: быстрая запись (2 минуты), прозрачное расписание, напоминания
- Для бизнеса: устранение овербукинга, снижение no-show с 15% до 5%, автоматизация записи

---

## 2. Глоссарий и роли

### Термины

| Термин | Определение |
|--------|-------------|
| **Слот** | Групповая тренировка: время, формат, инструктор, места |
| **Бронь** | Запись клиента на слот со статусом |
| **Прокатный фонд** | Доступное снаряжение (скальники + страховки) |
| **Дедлайн отмены** | 2 часа до начала тренировки |
| **Поздняя отмена** | Отмена после дедлайна (влечёт санкции) |
| **No-show** | Неявка на тренировку без отмены |

### Роли

| Роль | Описание | В скоупе MVP? |
|------|----------|---------------|
| **Клиент** | Посетитель скалодрома, записывается на тренировки | ✅ Да |
| **Инструктор** | Ведущий тренировок | ❌ Нет (админка) |
| **Администратор** | Владелец/сотрудник, управляет расписанием | ❌ Нет (админка) |
