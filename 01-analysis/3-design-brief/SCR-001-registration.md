# SCR-001: Регистрация / Вход

**Тип:** Экран  
**Зона:** НЗ (Неавторизованная)  
**Приоритет:** Must  
**Связанные FR:** FR-001, FR-002, FR-003, FR-004

---

## 1. Назначение
Экран авторизации клиента по номеру телефона через OTP-код. Обеспечивает быстрый вход без паролей.

## 2. Компоненты и иерархия
- **Логотип «Вертикаль»** (центрирован, сверху).
- **Заголовок:** «Вход в приложение».
- **Поле ввода телефона:** Маска `+7 (___) ___-__-__`.
- **Кнопка «Получить код»** (Primary, активна при валидном номере).
- **Поле ввода OTP:** 6 отдельных ячеек или одно поле с маской.
- **Таймер повторной отправки:** «Отправить код повторно через 00:59».
- **Кнопка «Подтвердить»** (на шаге OTP).

## 3. Поведение элементов
- При фокусе на поле телефона открывается цифровая клавиатура.
- После ввода номера кнопка «Получить код» становится активной.
- После нажатия «Получить код» — переход к полю OTP, запуск таймера.
- Автофокус на первой ячейке OTP. Автопереход между ячейками при вводе.
- При истечении таймера текст меняется на ссылку «Отправить код снова».

## 4. Состояния экрана
- **Default:** Поле телефона активно.
- **Loading:** Спиннер на кнопке после нажатия «Получить код» или «Подтвердить».
- **Error (неверный код):** Красная рамка у поля OTP, текст «Неверный код. Осталось попыток: N».
- **Error (превышен лимит):** Блокировка экрана на 15 минут с таймером (FR-003, NFR-008).

## 5. Копирайтинг
- Заголовок: «Добро пожаловать в Вертикаль!»
- Подзаголовок: «Введи номер телефона, и мы пришлём код».
- Ошибка формата: «Проверь номер телефона».
- Ошибка кода: «Код не подошёл. Попробуй ещё раз».

## 6. Связь с требованиями
- FR-001: Запрос OTP.
- FR-002: Проверка кода.
- FR-003: Rate-limit (таймер).
