# BS-004: Карта / Как добраться

**Тип:** Bottom Sheet / Modal  
**Зона:** АЗ  
**Приоритет:** Should  
**Связанные FR:** R-015

---

## 1. Назначение
Отображение адреса скалодрома и ссылки на внешние карты.

## 2. Компоненты и иерархия
- **Заголовок:** «Как добраться».
- **Текст адреса:** Крупно, с возможностью копирования по долгому тапу.
- **Кнопки внешних карт:**
  - «Открыть в Яндекс.Картах»
  - «Открыть в 2GIS»
  - «Открыть в Apple/Google Maps»
- **Текст-подсказка:** «Вход со стороны парковки, код домофона 1234».

## 3. Поведение элементов
- Открывается по тапу на «Как добраться» в SCR-003 или SCR-006.
- Тап на кнопку карты → открытие внешнего приложения (deep link).

## 4. Связь с требованиями
- R-015: Адрес текстом + ссылка на внешние карты (без встроенной карты для экономии ресурсов MVP).
