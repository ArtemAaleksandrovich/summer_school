# SCR-005: Мои записи

**Тип:** Экран  
**Зона:** АЗ  
**Приоритет:** Must  
**Связанные FR:** FR-018

---

## 1. Назначение
Список всех бронирований клиента с разделением на активные и прошедшие.

## 2. Компоненты и иерархия
- **AppBar:** Заголовок «Мои записи».
- **Сегментированный контроль (Tabs):** «Активные» / «Прошедшие».
- **Список броней:** Карточки, похожие на слоты, но со статусом.
- **Tab Bar:** Активная вкладка «Мои записи».

## 3. Карточка брони
- **Статус-бейдж:** «Подтверждена» (зелёный), «Отменена» (серый), «Отменена скалодромом» (красный).
- **Данные:** Дата, время, формат, инструктор.
- **Действие:** Кнопка «Подробнее» (ведёт на SCR-006).

## 4. Поведение элементов
- Переключение табов меняет список без перезагрузки экрана (фильтрация на клиенте или отдельный запрос).
- Pull-to-refresh для обновления статусов.

## 5. Состояния экрана
- **Empty (Активные):** «У тебя нет предстоящих тренировок. Самое время записаться!» + кнопка «К расписанию».
- **Empty (Прошедшие):** «Ты ещё не был на тренировках».

## 6. Связь с требованиями
- FR-018: Отображение списка броней.
