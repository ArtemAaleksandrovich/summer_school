# BS-001: Фильтры расписания

**Тип:** Bottom Sheet  
**Зона:** АЗ  
**Приоритет:** Must  
**Связанные FR:** FR-007, FR-008, FR-010

---

## 1. Назначение
Выбор параметров для фильтрации списка слотов на главном экране.

## 2. Компоненты и иерархия
- **Заголовок:** «Фильтры» + кнопка «Сбросить» (если есть активные фильтры).
- **Секция «Формат»:** Чекбоксы (мультивыбор).
  - [ ] Болдеринг
  - [ ] Трассы с верёвкой
- **Секция «Инструктор»:** Чекбоксы (мультивыбор, список из API).
- **Нижний CTA:** Кнопка «Применить» (фиксирована внизу шторки).

## 3. Поведение элементов
- Шторка открывается снизу на 50% высоты экрана.
- Выбор чекбоксов не закрывает шторку.
- Тап на «Применить» → закрытие шторки → обновление списка на SCR-002.
- Тап вне шторки → закрытие без применения.

## 4. Связь с требованиями
- FR-007: Фильтр по формату.
- FR-010: Семантика OR внутри группы.
