# SCR-007: Профиль

**Тип:** Экран  
**Зона:** АЗ  
**Приоритет:** Must  
**Связанные FR:** FR-005, FR-022, FR-023

---

## 1. Назначение
Просмотр данных аккаунта, статистики посещений и выход из системы.

## 2. Компоненты и иерархия
- **AppBar:** Заголовок «Профиль».
- **Блок пользователя:** Аватар (заглушка с инициалами), Имя, Номер телефона.
- **Блок статуса:** Если `is_regular == true` — золотой бейдж «Постоянный клиент 🏆».
- **Блок статистики:**
  - Всего посещений: N.
  - Поздних отмен: X / 3.
  - Неявок: Y / 2.
- **Блок уведомлений:** Тоггл «Push-уведомления» (управление локальными настройками).
- **Кнопка «Выйти из аккаунта»** (красная, внизу списка).
- **Tab Bar:** Активная вкладка «Профиль».

## 3. Поведение элементов
- Тап на «Выйти» → Alert «Точно выйти?» → очистка токенов → переход на SCR-001.

## 4. Связь с требованиями
- FR-005: Logout.
- FR-022: Отображение статуса постоянного клиента.
- FR-023: Отображение счётчиков санкций.
