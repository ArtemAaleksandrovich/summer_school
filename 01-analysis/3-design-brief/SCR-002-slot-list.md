# SCR-002: Список слотов (Главная)

**Тип:** Экран  
**Зона:** АЗ (Авторизованная)  
**Приоритет:** Must  
**Связанные FR:** FR-006, FR-007, FR-009, FR-010, FR-011

---

## 1. Назначение
Главный экран приложения. Отображает расписание тренировок на ближайшие 7 дней с возможностью фильтрации.

## 2. Компоненты и иерархия
- **AppBar:** Заголовок «Расписание», иконка фильтров (справа).
- **Горизонтальный скролл дат:** Чипы с датами (Пн 12, Вт 13...). Активная дата выделена.
- **Список слотов (вертикальный скролл):** Карточки слотов (SCR-003 preview).
- **Tab Bar:** Активная вкладка «Расписание».

## 3. Карточка слота (элемент списка)
- **Время:** Крупно, слева (например, `19:00`).
- **Формат:** Бейдж «Болдеринг» (синий) или «Трассы» (оранжевый).
- **Инструктор:** Имя + маленькое фото (аватар).
- **Места:** Текст «Свободно 4 из 16». Если мест < 3 — оранжевый текст «Мало мест!».
- **Прокат:** Иконка ботинка. Если есть — зелёная, если нет — серая с текстом «Нет проката».

## 4. Поведение элементов
- Тап на чип даты → фильтрация списка по дате.
- Тап на иконку фильтров → открытие Bottom Sheet (BS-001).
- Тап на карточку слота → переход на экран SCR-003.
- Pull-to-refresh → обновление списка.

## 5. Состояния экрана
- **Loading:** Скелетоны карточек (3-4 штуки).
- **Content:** Список карточек.
- **Empty:** Если нет слотов на выбранную дату/фильтр. Иллюстрация + «На эти дни тренировок пока нет».
- **Error:** «Не удалось загрузить расписание» + кнопка «Повторить».

## 6. Связь с требованиями
- FR-006: Показ на 7 дней.
- FR-007, FR-010: Фильтрация.
- FR-009: Отображение данных слота.
- FR-011: Empty state.
