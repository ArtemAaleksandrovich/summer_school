# BS-002: Успешная бронь

**Тип:** Bottom Sheet / Modal  
**Зона:** АЗ  
**Приоритет:** Must  
**Связанные FR:** FR-012, FR-026

---

## 1. Назначение
Подтверждение успешного создания брони.

## 2. Компоненты и иерархия
- **Иконка:** Большая зелёная галочка (анимация появления).
- **Заголовок:** «Ты записан!»
- **Текст:** «Ждём тебя [Дата] в [Время]. Напоминание придёт за 2 часа».
- **Кнопка:** «Посмотреть мои записи» (ведёт на SCR-005).
- **Кнопка (secondary):** «Закрыть» (возврат на SCR-002).

## 3. Поведение элементов
- Появляется автоматически после успешного ответа API (201 Created) с экрана SCR-004.
- Закрытие по тапу вне области или по кнопке.

## 4. Связь с требованиями
- FR-012: Успешное завершение операции.
- FR-026: Информирование о напоминании.
